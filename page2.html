Welcome, here are some linux bash commands to try on a raspberry pi

Bash Command Function Example
Man Shows a manual for the chosen command man more
Pwd Shows current whole file path Pwd
ls Shows all files/directories in the directory you are currently in. Can also show contents at a chosen file path ls sampledirectory
yes repeatedley prints input super quickly until ctrl+c is pressed yes Hello World!
cat displays contents of input file cat index.html
sleep locks screen for certain number of seconds. can be stopped using ctrl+c sleep 200
more similar to cat, but displays more information and metadata cat samplebashscript
w displays what user is currently logged in and what they are doing w
echo prints a given string or input on the line echo Hello World!
clear clear the terminal of all past commands clear
